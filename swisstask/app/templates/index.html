Automated Provisioning

Microservices Architecture

The Battle Logs: Surviving the Chaos

"The Windows Update Disaster"

Problem: A forced host OS update killed all 4 virtual nodes, causing a network desync and zombie pods (Unknown status).
Solution: Performed a surgical 'Vagrant Reload' to re-initialize host-only interfaces and leveraged K8s Self-Healing to respawn broken services.

"The Swap Memory Embargo"

Problem: Kubelet failed to start on Worker nodes because Linux Swap was active, breaking deterministic scheduling.
Solution: Disabled Swap permanently via fstab and Ansible automation, ensuring 100% stability for the Kubernetes engine.

"YAML Decoding Crisis"

Problem: Strict decoding errors in K8s manifests due to incorrect secretKeyRef mapping for the Database URL.
Solution: Refactored the manifest hierarchy and injected DB credentials through Kubernetes Secrets for maximum security.

"The Secure Tunnel Breakthrough"

Problem: Exposing a local virtual lab to the public internet without opening dangerous router ports.
Solution: Implemented Cloudflare Zero Trust Tunnels within the cluster, providing encrypted outbound-only global connectivity.
